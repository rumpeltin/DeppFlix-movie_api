DeppFlix

This project will be a REST API for an application called “DeppFlix”, which interacts with a database that stores data about different movies.
I'm building it to get familiar with Node.js and overall server-side programming.


Request HTTP Method Endpoint/URL Request Data Format Response Data Format
Return a list of ALL movies GET /movies None JSON object containing ALL movies
Return data about specific movie (by title) GET /movies/[title] None JSON object containing data about specific movie
Return data about a genre (by name/title) GET /movies/genres/[name/title] None JSON object containing data about specific genre
Return data about a director (by name) GET /movies/directors/[name] None JSON object containing data about specific director
Allow new users to register POST /user/registration JSON object containing user data JSON object containing given data & allocated username
Allow users to update their info (username) PUT /user/account None text message: [user info] has been updated
Allow users to add a movie to their favorites PUT /movies None text message: [movie] has been added
Allow users to remove a movie from their favorites DELETE /movies/favourites None text message: [movie] has been removed
Allow existing users to deregister DELETE /user/acccount None text message: your email has been removed
